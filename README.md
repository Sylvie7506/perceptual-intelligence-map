# Perceptual Intelligence Knowledge Graph

An interactive single-page knowledge graph that maps the full technology stack behind perceptual intelligence systems — from raw sensor input to emotional computing.

## What is this?

This project organizes the complex landscape of perceptual intelligence into five interconnected layers:

- **Input Layer** — Multisensory data acquisition through vision, motion, haptic, audio, and biometric sensors
- **Transmission Layer** — Edge computing, cloud platforms, and real-time protocols (5G, WebRTC, MQTT) that move data with minimal latency
- **Processing Layer** — AI-driven understanding including NLP, spatial cognition (SLAM), computer vision, and audio intelligence
- **Reality Layer** — How processed information reaches humans through VR, AR, MR, holographic displays, spatial audio, haptic feedback, and projection mapping
- **Empathy Layer** — Affective computing, emotion recognition, brain-computer interfaces, and psychological modeling

Each layer contains expandable modules with detailed technical explanations, core technologies, and relevant tools.

## How to use

Open `index.html` in any modern browser. No installation, no dependencies, no internet connection required.

- Click any layer on the landing page to expand its technology modules
- Use the sticky navigation bar to jump between layers
- Click any card in the detail sections to reveal in-depth technical breakdowns

## Background

This was built as a personal study project while exploring the intersection of information art design and perceptual computing. The goal was to create a clear, navigable map of how these technologies relate to each other — something I couldn't find in a single place elsewhere.
