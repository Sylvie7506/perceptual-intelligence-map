01
Input Layer

感官（输入层）— 多维传感器

如同人体的眼耳鼻舌身，通过各类传感器捕获物理世界的多维信息，将模拟信号转化为数字信号。

视觉传感器

Vision Sensors
RGB摄像头、ToF深度相机、双目/结构光、LiDAR、事件相机。

核心技术

  • RGB 摄像头：最基础的图像采集，AR人脸识别、手势追踪的输入源
  • ToF 深度相机：通过飞行时间测量深度，iPhone FaceID 的原理
  • 双目/结构光：模拟人眼立体视觉，用于空间重建和SLAM定位
  • LiDAR：激光点云扫描，精度极高，自动驾驶和AR环境理解的核心
  • 事件相机：只记录像素变化，超低延迟，适合高速运动场景
OpenCVRealSenseARKitLiDAR

惯性与运动传感器

IMU & Motion
加速度计、陀螺仪、磁力计（合称IMU），追踪设备和人体的姿态与运动。

核心技术

  • 6DoF 追踪：六自由度（位移XYZ + 旋转RPY），VR头显的基础能力
  • 传感器融合：融合多传感器数据，卡尔曼滤波消除漂移误差
  • 骨骼追踪：通过IMU阵列或视觉方案还原人体骨骼姿态
IMUAHRSKalman Filter6DoF

触觉与力传感器

Haptic & Force
压力、力矩、振动传感器，让系统感知物理接触和力反馈。

核心技术

  • 压阻/压电传感器：感知按压力度，Force Touch 的原理
  • 柔性电子皮肤：可穿戴柔性传感阵列，采集大面积接触分布
  • 力反馈执行器：振动马达 ERM/LRA，提供触觉反馈闭环
e-skinForce TouchHaptic Engine

声学与环境传感器

Audio & Environment
麦克风阵列、温湿度、气压、光线传感器，采集听觉与环境上下文。

核心技术

  • 麦克风阵列：多麦拾音 + 波束成形，定位声源方向
  • 超声波传感：近距离高精度测距，手势识别辅助
  • 环境光 & 气压：自动调节AR画面亮度、辅助高度定位
BeamformingMEMS MicSpatial Audio

生理信号传感器

Biometric Sensors
心率PPG、脑电EEG、肌电EMG、皮肤电导GSR，采集生理与情绪状态。

核心技术

  • PPG 光电容积：智能手表测心率原理，反映压力水平
  • EEG 脑电：采集头皮微弱电信号，注意力检测、脑机接口
  • EMG 肌电：检测肌肉电活动，免接触手势控制
  • GSR 皮电：皮肤电导变化反映情绪唤醒程度
BCIEEGEMGPPG
02
Transmission Layer

神经（传输层）— 边缘计算·平台即服务·网络协议

如同神经系统高速传递信号，将海量数据以低延迟、高可靠方式送达处理中心，同时提供弹性算力。

边缘计算

Edge Computing
MEC、雾计算、边缘AI推理、云渲染、CDN——将算力下沉到用户身边。

关键技术

  • MEC 多接入边缘计算：5G基站旁部署算力，延迟压缩到5ms以内
  • 雾计算：终端与云之间构建多级计算节点，按紧急程度分层处理
  • 边缘AI推理：端侧NPU直接跑轻量模型，手势识别无需上云
  • 云渲染/串流：云端GPU渲染3D画面，视频流推送给轻量终端
  • CDN 内容分发：3D资源缓存到全球边缘节点，减少加载延迟
MECCloudXREdge TPUCDN

平台即服务 PaaS

Platform as a Service
Serverless、容器化Docker/K8s、XR开发平台、CI/CD持续交付。

关键技术

  • Serverless / FaaS：函数即服务，按触发次数计费，无需预置服务器
  • 容器化 Docker / K8s：应用标准打包，弹性伸缩，任何环境一致运行
  • XR 开发平台：Unity Cloud、Unreal Pixel Streaming 等空间计算云服务
  • CI/CD：代码提交后自动测试、构建、部署，缩短迭代周期
AWS LambdaDockerK8sUnity Cloud

5G / 6G 移动通信

Mobile Network
eMBB大带宽、URLLC超低延迟、mMTC海量连接——XR无线化的核心。

三大场景

  • eMBB：下行峰值20Gbps，8K全景视频、云VR串流
  • URLLC：端到端1ms，远程手术、工业AR实时操控
  • mMTC：每km²百万终端，智能展厅数百传感器同时工作

6G愿景：太赫兹频段、通感一体、AI原生网络、空天地海覆盖。

实时通信协议

Real-time Protocols
WebRTC、WebSocket、MQTT、HTTP/3 (QUIC)、gRPC——数据传输的规则。

协议对比

  • WebRTC：浏览器间P2P音视频直连，多人AR协作常用
  • WebSocket：全双工长连接，服务器可主动推送实时状态
  • MQTT：发布/订阅轻量协议，物联网传感器首选
  • HTTP/3 (QUIC)：基于UDP多路复用，解决TCP队头阻塞
  • gRPC：高性能远程调用框架，微服务通信效率极高
WebRTCMQTTQUICgRPC

Processing Layer

大脑（处理层）— 语义理解·空间认知·深度学习

如同大脑对感官信号进行高级加工，利用AI技术对输入数据进行理解、推理和决策。

语义理解 NLP/NLU

Semantic Understanding
大语言模型LLM、意图识别、知识图谱、多模态理解——机器理解人类语言。

核心技术

  • 意图识别 + 槽填充：理解用户想做什么以及涉及哪些实体
  • 知识图谱：结构化世界知识网络，支持推理和问答
  • 多模态理解：同时理解文本、图像、音频，跨模态推理
TransformerBERTGPTRAG

空间认知与 SLAM

Spatial Cognition
V-SLAM视觉定位、3D重建NeRF、语义分割、空间锚点——设备理解环境。

核心技术

  • V-SLAM：摄像头实时构建环境地图并同步定位设备位置
  • 3D 重建：NeRF / Gaussian Splatting 从多角度还原三维场景
  • 语义分割：像素级识别场景物体类别（墙壁、桌子、人）
  • 空间锚点：物理空间标记虚拟坐标，多设备共享AR场景
SLAMNeRFARCoreSpatial Anchor

Deep Learning & CV
CNN/ViT图像特征、YOLO目标检测、姿态估计、生成式AI。

核心技术

  • CNN / ViT：卷积网络与视觉Transformer，图像特征提取双路线
  • YOLO 目标检测：实时定位识别画面中物体，AR标签叠加前提
  • 姿态估计：从图像推断人体/手部关键点，驱动虚拟化身
  • 生成式AI：Diffusion模型从文本生成图像、纹理、3D资产
PyTorchYOLOMediaPipeDiffusion

音频智能

ASR语音识别、TTS语音合成、声纹识别、音频事件检测。

  • ASR 语音识别：语音转文字，Whisper等端到端模型

Reality Layer

感知（现实层）— 扩展现实与多模态呈现

不只VR/AR/MR，还包括全息、空间音频、触觉反馈、投影映射——一切"让人感知数字信息"的技术。

VR 虚拟现实

Virtual Reality
完全沉浸式虚拟环境——HMD头显、Inside-Out追踪、注视点渲染、全身动捕。

核心技术

  • HMD 头显：双屏/Pancake光学，高分辨率低余晖近眼显示
  • Inside-Out 追踪：头显自带摄像头空间定位，无需外部基站
  • Foveated Rendering：眼动追踪+中心高分辨率渲染，降低GPU负载
  • 全身动捕：追踪手指、身体、面部表情，驱动虚拟化身
QuestSteamVRPancakeEye Tracking

AR 增强现实

Augmented Reality
虚拟信息叠加到真实世界——光波导、空间计算、遮挡处理、手机AR。

核心技术

  • 光波导：AR眼镜核心光学元件，将微型屏幕光导入人眼
  • 空间计算：理解物理环境几何和语义，让虚拟物体"落地"
  • 遮挡处理：虚拟物体被真实物体挡住时正确渲染遮挡关系
  • 手机AR：ARKit/ARCore让普通手机也能做AR
ARKitARCoreWaveguideSpatial Computing

MR 混合现实

Mixed Reality
虚实深度融合——空间映射、Passthrough、共定位、手势与眼动交互。

核心技术

  • 空间映射：实时扫描房间3D网格，虚拟球从真实桌面弹开
  • Passthrough：VR头显透视现实，在VR中看到真实环境
  • 共定位：多人在同一物理空间共享虚拟场景
  • 手势 & 眼动：用手抓取虚拟物体，用眼神选择目标
Vision ProHoloLensPassthrough

Empathy Layer

共情（高阶层）— 情感计算

最高层追求理解"感受是什么"——感知、识别、理解和回应人的情绪状态，实现人机共情。

情绪识别

Emotion Recognition
面部表情FER、语音情感、文本情感分析、生理信号情绪推断。

核心技术

  • FER 面部表情：基于面部动作单元AU编码，识别微表情和情绪类别
  • 语音情感：从语调、语速、能量等声学特征推断情绪
  • 文本情感：NLP情感极性判断，舆情和用户反馈分析
  • 生理信号：HRV心率变异性、GSR皮电等客观指标判断唤醒水平

情感生成与表达

Affective Generation
情感语音合成、虚拟形象表情驱动、情感适应性交互、共情对话。

核心技术

  • 情感TTS：在语音合成中注入情绪参数，让合成语音有喜怒哀乐
  • Blendshape驱动：将情感状态映射为数字人面部动画
  • 情感适应：根据用户情绪实时调整对话策略、UI氛围或音乐
  • 共情对话：大模型结合心理学模型展现理解和关怀
